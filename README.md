# Suman Karmakar Portfolio

Personal portfolio website built with HTML, CSS, and JavaScript.

## Overview

This project showcases:
- Hero section with personal introduction and photo gallery
- Featured projects with live demo/source links
- About and Tech Stack sections
- Education timeline with semester result previews
- Certifications gallery
- Contact section and social links
- Light/Dark mode and smooth UI interactions

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Devicon + Font Awesome (icons)

## Project Structure

```text
Portfolio/
|- index.html
|- style.css
|- script.js
|- README.md
|- Suman_karmakar_Resume.pdf
`- photos/
   |- images/         # Hero/personal photos
   |- pictures/       # Project preview screenshots
   |- certificates/   # Certificate images
   `- results/        # Academic result images
```

## Run Locally

1. Clone/download this repository.
2. Open the folder in terminal.
3. Start the local server:

```bash
npm run dev
```

4. Open `http://127.0.0.1:5500` in your browser.

Alternative (without Node scripts): open `index.html` directly or use Live Server.

## Customization

- Update text/content in `index.html`
- Update styles in `style.css`
- Update interactions/animations in `script.js`
- Replace media inside `photos/` and keep file paths updated in `index.html`
